Platform: Windows
Build: MonoGame / C#

Puzzle Kings is an updated version of the classic ’15 Puzzle’ game. You must reorder the randomly shuffled tiles of the numbers 1 – 15 back into the correct numerical order. Not only is it challenging, but with the addition of timers and higher grid sizes, the pressure is really on!

January 2016 – February 2016 / 4 weeks

This was the first project I developed using C# ( and MonoGame ), it was very interesting to see how different it was to program in compared to Javascript. This project was really a primer into object-oriented programming, coming from a more functional language like JavaScript. Although it was only 4 weeks long, we had to learn quickly, as the purpose of this project was to get to grips with C# in preparation of building a Unity game.

I found this project a lot of fun to work with, particularly as before this I'd been so used to working without a framework. Having MonoGame around really let me take a look into what I had been doing right and wrong when developing my own basic systems, as well as allowing me to focus on getting the game logic correct.
