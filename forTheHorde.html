Platform: Web
Build: HTML5 / JavaScript

For the Horde! is a text-based RPG inspired by works such as the ‘Fighting Fantasy’ books and ‘The Hobbit’. Based in Blizzard’s ‘Warcraft’ universe, you play as a young Orc hunter seeking to gain fame and reputation as a daring adventurer.

September 2015 – November 2015 / 8 weeks

For The Horde! was written in JavaScript on top of the HTML5 canvas. I didn’t use any frameworks to build this, it was written completely from scratch. I did, however, use the libraries jQuery and SweetAlert ( for the nice window popups ).

I used a data-driven design to handle the dialogue and entities, importing NPCs, locations and items from multiple JSON files where they were parsed and accessed depending on the game state and player’s current status. I was pleased with the way I handled this system, as it was my first time working with data files.

This was definitely a difficult project for me. Although I had a little experience in games development at this point, I had focused wholly on just getting a product working without caring about the code architecture. However, with this I spent a lot of time looking into the best ways to handle certain systems, so I really felt I grew as a games developer.
