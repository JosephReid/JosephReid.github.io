Platform: Web
Build: HTML5 / JavaScript

Counting Sheep is a Flappy Bird inspired arcade side-scroller. You play as a sheep stuck in DreamLand, the only way out is for you to wake up! In a strange reverse of fate, count as many sheep in the dream as you can to fall asleep and escape! Experience my cute take on the genre.

November 2015 – December 2015 / 4 weeks

This was a game written purely in Javascript on top of the HTML5 canvas from scratch. It was interesting to work without a framework on a project like this, as there are a lot of aspects of game programming, such as sprites, animations and audio, that are usually handled by a framework on a lower level. This meant I had to write some of these basic features myself.

This was a tough one as I hadn’t handled drawing animated sprites before in JS / HTML5. I had to write my own small system to load the images in from files and generate Sprite / Animation objects. I also had to experiment with writing a lot of visual effects, such as parallax scrolling, text / sprite fading and particles. I also had fun writing some procedural generation, for the obstacles that spawn and for the background.
